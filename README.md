# 🧠 AI Mental Health Emotion Analyzer

A full-stack Machine Learning project that analyzes user text and predicts emotions using NLP, with real-time API, database storage, and interactive dashboard.

---

## 🚀 Features

- 🔮 Emotion prediction using NLP (Logistic Regression + TF-IDF)
- ⚡ REST API using Flask
- 🗄️ PostgreSQL database integration
- 📊 Streamlit dashboard for visualization
- 📜 History tracking of user inputs
- 🗑️ Clear history functionality
- 📈 Emotion distribution charts

---

## 🧱 Tech Stack

- Python
- Machine Learning (Scikit-learn)
- NLP (NLTK)
- Flask (Backend API)
- PostgreSQL (Database)
- Streamlit (Frontend Dashboard)
